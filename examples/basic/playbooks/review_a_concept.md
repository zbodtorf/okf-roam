# Steps

1. Confirm the document has a non-empty `type`.
2. Check that its links resolve within the [Knowledge Bundle](/concepts/knowledge_bundle.md).
3. Compare it with the [OKF Specification](/references/okf_specification.md).
4. Follow the [Publishing Guide](/playbooks/publish_a_bundle.md).
