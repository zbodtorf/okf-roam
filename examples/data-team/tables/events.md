# Schema

| Column | Type | Description |
|---|---|---|
| `event_id` | string | Unique event identifier. |
| `user_id` | string | User that generated the event. |
| `event_name` | string | Normalized event name. |
| `occurred_at` | timestamp | Time the event occurred in UTC. |

# Joins

Join to [Users](/tables/users.md) on `user_id`.
